REVIEW Dependencies - Detailed Analysis

reqwest 0.12

Capability: Full HTTP client with TLS support

Usage: All Google API requests - Discovery Document fetch, API execution, OAuth token exchange

Features: json, stream, rustls-tls (default features disabled - no OpenSSL)

Assessment: Essential for a Google API CLI. Uses rustls for TLS (pure Rust, no system dependency). Only connects to googleapis.com domains.

yup-oauth2 12

Capability: OAuth2 token exchange, service account JWT signing

Usage: Token refresh, service account auth, installed app OAuth flow

Assessment: Standard Google OAuth library for Rust. Only communicates with Google OAuth endpoints.

keyring 3.6.3

Capability: Read/write OS credential store

Usage: Stores AES-256-GCM encryption key in macOS Keychain / Windows Credential Manager / Linux Secret Service

Assessment: Security best practice for credential storage. Falls back to file-based storage if keyring unavailable.
