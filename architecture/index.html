What This Project Does

gws is a command-line tool for interacting with Google Workspace APIs (Gmail, Drive, Calendar, Sheets, Docs, Chat, etc.). Instead of hardcoding API definitions, it fetches Google's Discovery Documents at runtime and dynamically builds its CLI command tree.


Key behaviors:

  • Authenticates via OAuth2 (user credentials) or Service Account (ADC)
  • Stores credentials encrypted with AES-256-GCM in ~/.config/gws/
  • Caches Discovery Documents locally (24h TTL)
  • Sends authenticated HTTP requests to Google APIs only
  • Supports file upload (Drive) and email sending (Gmail)
  • Can run as an MCP server for Claude/LLM integration
  • Has a TUI wizard for OAuth scope selection

What it does NOT do: No telemetry, no auto-updates, no daemon mode, no analytics, no crash reporting, no data collection, no third-party service calls.
